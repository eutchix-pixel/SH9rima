Pour intégrer cette dimension spatio-temporelle à votre application Replit, vous avez besoin d'un module de **cartographie interactive synchronisé avec une frise chronologique**.

Voici le "package" complet (instructions techniques + structure de données) à copier-coller dans votre interface Replit AI pour générer cette fonctionnalité.

---

## Instructions pour Replit AI

**Objet : Ajout d'un module "Carte Temporelle Interactive" pour la page Naissance du 9.**

### 1. Objectif Technique

Créer un composant React/Vue (selon l'architecture) nommé `GlobalMapChronology` qui combine une carte du Tonkin et une frise chronologique. Le déplacement sur la frise doit mettre à jour les points d'intérêt (POI) sur la carte.

### 2. Stack recommandée

* **Cartographie :** Leaflet.js ou React-Leaflet (léger, adapté au style "carte ancienne").
* 
**Style :** Utiliser un filtre CSS `sepia(0.8) contrast(1.2)` sur les tuiles de carte pour correspondre au style "musée colonial". 


* **Interactivité :** Framer Motion pour les transitions fluides entre les dates.

### 4. Spécifications d'Interface (UI/UX)

* **Frise Chronologique :** Placer une barre horizontale en bas de la carte. Chaque clic sur une date doit centrer la carte sur le marqueur correspondant avec un zoom fluide. 


* 
**Popups Immersifs :** Lorsqu'un point est sélectionné, afficher un "cartel" de musée avec le texte descriptif et, si disponible, l'image d'archive associée (ex: Croquis de Dongly). 


* 
**Calques Géographiques :** Ajouter des tracés de couleur bleue pour souligner le **Fleuve Rouge**, la **Rivière Noire** et la **Rivière Claire**, car "la carte commande le combat".
